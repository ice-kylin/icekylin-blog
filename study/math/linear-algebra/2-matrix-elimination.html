# 高斯消元法

# 🪶 高斯消元法的历史

该方法以德国数学家高斯（Gauss）命名，由拉布扎比·伊丁特改进，发表于法国但实际上最早出现于中国古籍《九章算术》，成书于约公元前 150 年。

在线性方程组中的消元法和矩阵的消元法本质上是一样的。

# 消元失效

正如线性方程组可能出现无解或多解的情况，矩阵消元可能会失效。规则也很简单 —— 主元不能为 。

这也不难理解，类似线性方程组的求解，如果某一个方程有一个还未计算数值的未知数和其它 0 或多个已计算数值的未知数，而还未计算数值的未知数的系数为 ，则无法算出此未知数的具体数值。

回到矩阵消元，如果恰巧消元至某行时 出现在了应该是主元的位置上，又因为主元不能为 ，此时可尝试通过行交换的方式解决 —— 也就是将下方的某行和此行交换，使得非 数字出现在主元位置上使得消元得以继续。但如果 出现在主元位置上且下方没有可供交换的行，则消元终止，并证明矩阵 为不可逆矩阵（也叫奇异矩阵，在之后还将详细讨论），且线性方程组没有唯一解。例如消元得到 或 （注意标注橙色的位置）。
